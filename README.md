# TK Portfolio

Personal portfolio website built with static HTML, CSS, and JavaScript.

- Live (GitHub Pages): https://TKylmz.github.io/TK-Website/
- Sources: This repository

Quick start

1. Open `index.html` in your browser for a quick preview.
2. Or run a simple local server:

```bash
python -m http.server 8000
# then open http://localhost:8000
```
